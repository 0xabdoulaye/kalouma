Politique des cookies

Derniere mise a jour : 9 mars 2026

Resume : Notre application mobile n'utilise pas de cookies. Ce site web (kalouma.com) utilise uniquement un outil d'analyse anonyme pour ameliorer nos services. Aucune donnee personnelle n'est collectee via des cookies.

1. Application mobile

L'application Kalouma Market n'utilise pas de cookies. Les donnees sont stockees localement sur votre appareil de maniere securisee pour :

  • Session de connexion : votre token d'authentification pour rester connecte
  • Cache local : annonces et categories pour une navigation plus rapide
  • Preferences : vos parametres de notification et filtres de recherche

Ces donnees sont stockees sur votre appareil et ne sont pas partagees avec des tiers. Elles sont supprimees lorsque vous vous deconnectez ou desinstallez l'application.

2. Site web (kalouma.com)

Notre site web utilise les technologies suivantes :

PostHog Analytics

Nous utilisons PostHog pour comprendre comment les visiteurs interagissent avec notre site web. PostHog collecte des donnees anonymisees telles que :

  • Pages visitees
  • Clics sur les boutons de telechargement
  • Questions FAQ consultees
  • Type d'appareil et navigateur (de maniere anonyme)
Configuration respectueuse de la vie privee : L'autocapture est desactivee, les replays de session ne sont pas enregistres, et seules les vues de pages et les interactions essentielles sont suivies.
Technologie Objectif Duree
PostHog Analyse d'audience anonyme Session
Google Fonts Affichage des polices de caracteres Cache navigateur
Font Awesome (CDN) Affichage des icones Cache navigateur

3. Cookies tiers

Nous n'utilisons aucun cookie publicitaire ni de suivi comportemental. Nous ne vendons ni ne partageons vos donnees de navigation avec des annonceurs.

4. Vos choix

Vous pouvez a tout moment :

  • Bloquer les cookies dans les parametres de votre navigateur
  • Utiliser le mode navigation privee pour empecher le stockage local
  • Desactiver JavaScript pour empecher PostHog de fonctionner (note : cela peut affecter l'experience du site)

Le blocage des technologies ci-dessus n'affectera pas votre utilisation de l'application mobile Kalouma Market.

5. Contact

Pour toute question concernant notre utilisation des cookies et technologies similaires :

  • Email : staff@kalouma.com
  • Site web : https://kalouma.com
